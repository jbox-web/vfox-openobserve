# vfox-openobserve

[![CI](https://github.com/jbox-web/vfox-openobserve/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-openobserve/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [openobserve](https://github.com/openobserve/openobserve)

## Install

```
mise plugin install openobserve https://github.com/jbox-web/vfox-openobserve.git
```
